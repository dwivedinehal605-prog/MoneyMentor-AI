# MoneyMentor AI

## Tagline
**Spend Smarter. Save Better. Build Your Future.**

## Overview

MoneyMentor AI is an AI-powered personal finance and investment platform designed to help users manage their finances efficiently. It enables users to track expenses, monitor income, create budgets, analyze spending patterns, and receive AI-driven financial insights and personalized investment recommendations.

## Key Features

- Expense Tracking
- Income Management
- Budget Planning
- AI Spending Analysis
- Investment Recommendations
- Financial Dashboard
- Savings Goal Tracking
- Reports & Analytics

## Technology Stack
